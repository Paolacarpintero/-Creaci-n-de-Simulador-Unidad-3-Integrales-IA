Contenido 3.1 - Integral indefinida y antiderivadas

La integral indefinida consiste en encontrar una función cuya derivada sea la función dada. Se representa como:

$$\int f(x)\,dx=F(x)+C$$

Propiedades

• La derivada y la integral son operaciones inversas.

• Siempre aparece la constante de integración C.

• La integral es lineal.

Fórmulas básicas

$$\int x^n dx=\frac{x^{n+1}}{n+1}+C \qquad n\neq -1$$

$$\int \frac1x dx=\ln|x|+C$$

$$\int e^x dx=e^x+C$$

$$\int \sin(x)dx=-\cos(x)+C$$

$$\int \cos(x)dx=\sin(x)+C$$

Ejemplo

Resolver:

$$\int 4x^3dx$$

Resultado:

$$x^4+C$$

Contenido 3.2 - Métodos de integración

1. Sustitución

Se utiliza cuando existe una función dentro de otra.

$$u=x^2+1$$

$$du=2xdx$$

$$\int2x(x^2+1)^5dx$$

Resultado:

$$\frac{(x^2+1)^6}{6}+C$$

2. Integración por partes

$$\int udv=uv-\int vdu$$

Se utiliza cuando existe un producto entre funciones.

$$\int xe^xdx$$

Resultado

$$xe^x-e^x+C$$

3. Fracciones parciales

Cuando la función es racional.

$$\frac{3x+5}{(x+1)(x+2)}$$

4. Integrales estándar

$$\int\frac1{a^2+x^2}dx=\frac1a\tan^{-1}\left(\frac xa\right)+C$$

Contenido 3.3 - Integral definida y Teorema Fundamental del Cálculo

Una integral definida permite calcular áreas bajo una curva.

$$\int_a^b f(x)dx$$

Teorema Fundamental

Si

$$F'(x)=f(x)$$

Entonces

$$\int_a^b f(x)dx=F(b)-F(a)$$

Ejemplo

$$\int_1^3 2xdx$$

Antiderivada:

$$x^2$$

Resultado:

$$9-1=8$$

Contenido 3.4 - Integrales impropias

Las integrales impropias aparecen cuando:

• Uno de los límites es infinito.

• La función presenta discontinuidades.

Ejemplo 1

$$\int_1^\infty \frac1{x^2}dx$$

Resultado:

$$1$$

Convergente.

Ejemplo 2

$$\int_1^\infty\frac1x dx$$

Resultado:

Divergente.
